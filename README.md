### Instalação
Download the repository:
```
git clone https://github.com/albertosilv/F1R3NZ3-Desafio-Mobile.git
```
Enter in folder:
```
cd F1R3NZ3-Desafio-Mobile
```
Then, install all dependencies with yarn or NPM:
```
npm i
```
or
```
yarn
```
